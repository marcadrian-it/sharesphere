# Stack

## Shared

- TypeScript
- Apollo GraphQL/URQL
- NodeJS v18
- Yarn v1.22.19

## Backend

- Express
- Redis
- TypeORM
- PostgreSQL
- Argon2
- Docker/Dokku

## Frontend
- Vercel
- NextJS
- ChakraUI
- GraphQL Code Generator

## CDN
- Cloudinary

## Instructions
Sync schema directly from dist folder
Example: npx typeorm schema:sync -d ./dist/data-source.js
